# Changelog

All notable changes to Visual Media Layout are documented here.

## 0.1.0 - 2026-07-01

### Added

- Initial public release preparation.
- Visual media layout code block renderer.
- Reading View media layout support.
- Live Preview media layout support.
- Image and video rendering for vault and web media.
- Automatic 1-4 column row layout.
- Alignment, delete, resize, and drag interactions.
- Commands for inserting empty layouts and converting Markdown media.
- Automatic media-only line conversion.
- Adjacent layout block merging.
