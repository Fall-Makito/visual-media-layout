# Security Policy

## Official Downloads

Only download Visual Media Layout from:

- The official GitHub Releases page for this repository

Do not install builds from cloud drive links, chat attachments, reposted zip files, or third-party mirrors.

## Reporting a Vulnerability

Please do not open a public issue for security problems.

Send a private report to:

```text
rfmakito@163.com
```

Include:

- Plugin version
- Obsidian version
- Operating system
- Steps to reproduce
- What data or behavior may be affected

## Release Integrity

Release builds include a `sha256sums.txt` file in `dist/visual-media-layout/`.

For GitHub Releases, attach the official files directly to the release:

- `main.js`
- `manifest.json`
- `styles.css`
