# Visual Media Layout 0.1.0

Initial public release.

## Highlights

- Arrange images and videos into visual layout blocks in Obsidian.
- Use 1-4 columns per row with consistent row height.
- Convert existing Markdown media embeds into layout blocks.
- Align, delete, resize, and reorder media items.
- Use Reading View for the most complete editing experience.

## Installation

Download these files from this release:

- `main.js`
- `manifest.json`
- `styles.css`

Place them in:

```text
YourVault/.obsidian/plugins/visual-media-layout
```

Restart Obsidian and enable Visual Media Layout in Obsidian settings.

## Compatibility

- Obsidian: 1.5.0 or newer
- Plugin ID: `visual-media-layout`
- Version: `0.1.0`
