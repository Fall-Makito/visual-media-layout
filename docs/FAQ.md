# FAQ

## What is Visual Media Layout?

Visual Media Layout is an Obsidian plugin for arranging images and videos in visual rows inside notes.

## Does it upload my vault data?

No. The plugin is designed to work inside your Obsidian vault.

If your note references web images or web videos, Obsidian may request those URLs so the media can be displayed.

## Why use GitHub Releases instead of cloud drive links?

GitHub Releases give each build a clear version, source, history, and checksum. Cloud drive links are harder for users to trust and harder for you to maintain.

## Which mode works best?

Reading View is currently the most reliable mode for drag-heavy editing. Live Preview is usable, but some save paths depend on the information Obsidian provides to plugins.

## What files do users need for manual install?

Users need:

- `main.js`
- `manifest.json`
- `styles.css`

These should come from the official GitHub Release.

## Will this be listed in Obsidian Community Plugins?

No. The planned public distribution path is GitHub Releases.
