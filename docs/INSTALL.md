# Install Visual Media Layout

## Install From GitHub Releases

GitHub Releases is the official distribution path for Visual Media Layout.

1. Open the latest GitHub Release.
2. Download:
   - `main.js`
   - `manifest.json`
   - `styles.css`
3. Open your Obsidian vault folder.
4. Create this folder:

```text
.obsidian/plugins/visual-media-layout
```

5. Move the three downloaded files into that folder.
6. Restart Obsidian.
7. Open Settings, Community plugins, and enable Visual Media Layout.

## Compatibility

| Item | Requirement |
| --- | --- |
| Obsidian | 1.5.0 or newer |
| Desktop | Supported |
| Mobile | Not marked desktop-only |

## Do Not Use Cloud Drive Links

Cloud drive links are hard for users to verify and hard for maintainers to version properly.

Use the official GitHub Releases page instead.
